# Ejercicio 2: Lista de Tareas con DOM y LocalStorage

Aplicación funcional de gestión de tareas con persistencia de datos.

## Puntos clave
- Uso de addEventListener para el formulario.
- Creación y eliminación de nodos en el DOM.
- Guardado en LocalStorage mediante JSON.stringify.
- Carga automática de tareas al iniciar.
